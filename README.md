# Clonando a Página do YouTube com CSS

**Projeto de Estudo** feito durante o desafio da DIO, com o objetivo de **clonar a página do YouTube** utilizando apenas **HTML e CSS**, colocando em prática os conceitos de **Flexbox**.

> ⚠️ Este projeto é **apenas para fins de estudo** e **não é oficial do YouTube**.

## Objetivo do Projeto

Neste desafio, esta sendo usado:

- **Flexbox** e layout responsivo.
- Clone da interface do YouTube a partir de um **modelo no Figma**.
- Criatividade e **identidade visual** ao projeto.
